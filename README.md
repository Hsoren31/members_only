## About the Project

This project is a members-only clubhouse application where users can create and share anonymous posts. While all visitors can view the messages, only authenticated members of the clubhouse can see the identities of the authors, creating a layered access experience based on user roles.

The application focuses on implementing secure authentication, role-based authorization, and relational data management. Users can sign up, log in, and create posts, while membership status and admin privileges determine what information and actions are available to them.

  ### Built with
  - NodeJS
  - PostgreSQL
  - Passport
  - Express
